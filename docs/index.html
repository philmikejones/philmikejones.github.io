Phil Mike Jones¶

I write code to analyse spatial and non-spatial data sets. I typically use Python, R, and Linux. I’ve written a number of tutorials on these topics that were originally to remind me how to do things, but which you’re welcome to use if they’re any help to you.

You can use and share these tutorials for free (CC BY-NC). I try to ensure they’re as accurate as possible and hope they’re helpful, but make no guarantees that they’re correct; use at your own risk. Always be careful copying and pasting code from an internet source before running it on your machine.
